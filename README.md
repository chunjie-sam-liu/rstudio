# Rstudio configs

Gather the rstudio self defined configurations.

## Shortcut example
```json
{
    "fold": "Cmd+[",
    "foldall": "Alt+Cmd+[",
    "unfold": "Cmd+]",
    "unfoldall": "Alt+Cmd+]",
    "removeline": "Ctrl+Backspace"
}
```

```json
{
    "fold": "Cmd+[",
    "foldAll": "Alt+Cmd+[",
    "unfold": "Cmd+]",
    "unfoldAll": "Alt+Cmd+]",
    "goToFileFunction": "Cmd+.",
    "commentUncomment": "Cmd+/"
}
```
